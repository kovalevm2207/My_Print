Ссылка на файл с описанием NASM:

~~~~~~~~~~~~~~~~~~~~~
| Флаги компиляции: |
~~~~~~~~~~~~~~~~~~~~~
-w+orphan-labels                                                        Приведет к предупреждению если нет ':' после метки
-o:                                                                     Имя выходного файла
-f:                                                                     Формат выходного файла
-l:                                                                     Создание файла с описанием объекта недвижимости.
-L:                                                                     Дополнительная или измененная информация об объекте недвижимости.
-M:                                                                     Сгенерировать зависимости Makefile
-MG:                                                                    Сгенерировать зависимости Makefile
-MF:                                                                    Установить файл зависимостей Makefile
-MD:                                                                    Сборка и генерация зависимостей
-MT:                                                                    Имя целевого объекта зависимости
-MQ:                                                                    Имя целевого объекта зависимости (в кавычках)
-MP:                                                                    Создание фиктивных целей Makefile
-mW:                                                                    Стиль цитирования -MWWatcommake
-F:                                                                     Формат отладочной информации
-g:                                                                     Включение отладочной информации.
-X:                                                                     Выбор формата отчета об ошибках
-Z:                                                                     Отправлять ошибки в файл.
-s:                                                                     Отправлять сообщения об ошибках вstdout
-i:                                                                     Включить каталоги поиска файлов
-p:                                                                     Предварительное включение файла
-d:                                                                     Предварительное определение макроса
-u:                                                                     Отменить определение макроса
-E:                                                                     Только предварительная обработка
-a:                                                                     Отключить предварительную обработку
-O:                                                                     Многопроходная оптимизация
-t:                                                                     Режим совместимости с TASM
-w:                                                                     Включить или отключить предупреждения сборок -w.-W
-v:                                                                     Отображение информации о версии

~~~~~~~~~~~~
| Правила: |
~~~~~~~~~~~~

'\'                                                                     Используется для переноса длинной строки с командой

## Допустимые символы для меток:

'_', '$', '#', '@',
'.', '?', буквы и цифры

## Префиксы инструкций:

REP
LOCK
REPE/REPZ
REPNE/REPNZ
XACQUIRE/XRELEASE
BND/NOBND

## Префиксы размера адреса и операнда
byte, word, dword
A16, A32, A64
far                                                                     Для косвенных дальних вызовов или переходов

## Псевдоинструкции

DB, DW, DD, DQ, DT, DO, DY и DZ                                         Инициализация данных соответствующего типа

RESB, RESW, RESD, RESQ, REST, RESO, RESY и RESZ                         Объявление неинициализированных данных

INCBIN (сколько пропустить байт) (сколько максимум считать)             Включение внешних двоичных файлов, без аргументов считает весь файл

EQU                                                                     Определение констант

TIMES (сколько повторить) (что повторить)                               Повторение инструкции или данных;
(буфер: база данных 'hello, world'
        times 64-$+buffer db ' ')

0h `<N>` 0x                                                              Шестнадцатеричная
0d `<N>` 0t                                                              Десятеричная
0q `<N>` 0o                                                              Восьмеричная
0b `<N>` 0y                                                              Двоичная


Символы Юникода, указанные с помощью \u`or`, \Uпреобразуются в UTF-8. Например, следующие строки эквивалентны:

      db `\u263a` ; UTF-8 смайлик

$                                                                       Возвращает смещение
$$                                                                      Возвращает начало текущего сегмента

SEG                                                                     Возвращает предпочтительный сегмент символа
        mov     ax,seg symbol
        mov     es,ax
        mov     bx,symbol

.label                                                                  Локальная метка, относительно предыдущей нелокальной
label1.label                                                            Чтобы прыгать на конкретную, локальную метку, а не локальную метку этого сегмента кода
..@                                                                     Метка для макросов, не влияет на механизм работы локальных и нелокальных меток

jmp $                                                                   Трюк с бесконечным циклом

jmpabs target                                                           64-bit absolute jump
